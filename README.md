### dein

1. ```$ mkdir -p ~/.cache/dein```

2. ```$ cd ~/.cache/dein```

3. ```$ curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh```

4. ```$ sh ./installer.sh ~/.cache/dein```
